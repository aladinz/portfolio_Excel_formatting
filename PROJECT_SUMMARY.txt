YOUR PYTHON SCRIPTS (What Each Does)
====================================

restructure_type_b.py [NEW]
─────────────────────
Purpose: Convert Type B (single Data sheet) to Type A (professional structure)
Input: Excel file with Data sheet containing months as columns
Output: File with Executive Summary, Monthly Performance, Data Source sheets
Features:
  • Auto-detects month columns
  • Calculates KPIs (growth, best/worst month, averages)
  • Creates professional Executive Summary with 9 metrics
  • Organizes monthly data in color-coded sections
  • Applies professional formatting throughout
  • Fully automated - no manual data entry needed

Usage: python restructure_type_b.py "file.xlsx"


format_all.py [UNIVERSAL]
─────────────────────
Purpose: Detect file type and apply appropriate professional formatting
Input: Type A or Type B Excel files
Output: Professionally formatted files with consistent styling
Features:
  • Auto-detects Type A vs Type B structure
  • Applies color scheme (blues, yellows, greens)
  • Optimizes column widths
  • Adds borders and alignment
  • Works with single or multiple files
  • Batch processing support with wildcards

Usage: python format_all.py "file.xlsx"  or  python format_all.py *.xlsx
