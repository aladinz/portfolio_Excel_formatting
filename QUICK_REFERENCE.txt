TROUBLESHOOTING
===============

Solution: Use full path:
  C:\Users\aladi\Portfolio_Excel_Formattin\.venv\Scripts\python.exe format_all.py file.xlsx

Solution: Make sure the Excel file is in the same folder as format_all.py

Solution: Charts are normal - they're embedded. Just open the file in Excel
          and navigate to Executive Summary sheet to see them

Solution: Close the file in Excel, run formatter again, then open it fresh
